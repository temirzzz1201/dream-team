# Feedback Form App

Простой проект формы обратной связи на **Vue 3**, использующий **Pinia** для управления состоянием и кастомные UI-компоненты.

---

## 🛠 Технологии

- **Vue 3** – фронтенд фреймворк на Vue 3
- **Pinia** – глобальное состояние
- **TypeScript** – типизация компонентов
- **SCSS** – стили
- Кастомные компоненты:
  - `UiInput`, `UiTextarea` – стандартные поля формы
  - `UiButton` – кнопки
  - `AppSelect` – селект с кастомными опциями
  - `ErrorScreen` - экран ошибки
  - `FeedbackForm` - компонент формы обратной связи
  - `RatingStars` – рейтинг звездочками
  - `QuickReplies` – быстрые ответы
  - `FormTitle` – заголовки форм
  - `StepOne` - первый шаг формы
  - `StepTwo` - второй шаг формы
  - `SuccessScreen` - экран успешной формы

---

## ⚡ Функционал

1. Двухшаговая форма обратной связи:
   - **Шаг 1:** ФИО, Email, Телефон, Грейд
   - **Шаг 2:** Рейтинг, уровень, быстрые ответы, комментарий
2. Валидация полей с отображением ошибок
3. Поддержка мобильной версии через адаптивный grid
4. Кнопки «Назад», «Далее» и «Отправить» с эмитом событий
